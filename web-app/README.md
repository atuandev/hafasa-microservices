# Hafasa Frontend

## Getting Started

1. Download dependencies:

```bash
bun i
```
> install bun if don't have: `npm i bun -g`

2. Run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
